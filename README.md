# OpenSpot – Smart Parking (Java + MySQL)

OpenSpot helps list parking spots, check availability, and create/cancel simple reservations.

## Stack
- Java 17+, Spring Boot (Web, JDBC), MySQL 8
- Docker (for local DB)
- OpenAPI spec in `api/openapi.yaml`
